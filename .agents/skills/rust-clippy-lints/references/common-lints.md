# Common Lints

## Performance

- `redundant_clone` - Unnecessary cloning
- `large_enum_variant` - Large enum variant
- `needless_collect` - Unnecessary collection
- `iter_next_loop` - Inefficient loop

## Style

- `useless_format` - Useless format!
- `default_trait_access` - Default trait access
- `unnecessary_lazy_evaluation` - Unnecessary closure

## Correctness

- `eq_op` - Equal operands
- `unreachable` - Unreachable code
- `unused_self` - Unused self

## Complexity

- `too_many_arguments` - Too many arguments
- `type_complexity` - Complex type
